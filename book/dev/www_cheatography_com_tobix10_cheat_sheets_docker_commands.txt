Stored containers & images

Container basics

List running containers

docker ps

Run in a new named

docker run -it --name cont_name #image

List all containers

docker ps -a

container

#command

List images

docker images [--tree]

Run as a daemon

docker run -d #image #command

Remove container

docker rm #container

Start container

docker start -i #container

Remove image

docker rmi #image

Stop container

docker stop #container

Remove all containers

docker rm `docker ps --no-trunc -aq`

Restart container

docker restart #container

Remove all images

docker rmi `docker images -q`

Kill container

docker kill #container

Run a command

docker exec -it #container #command

#container - name or id
#image - name

#image - name

Logs & info
Read container stdout

docker logs [-f] #container

Show running processes

docker top #container

Show container configuration

docker inspect #container

Show resources usage

docker stats #container

Show changes in filesystem

docker diff #container

Show image history

docker history #image

Show container events

docker events

Import & Export
Export container to tar

docker export -o file.tar #container

Import container

docker import file.tar

Export image to tar

docker save -o file.tar #image[:tag]

Import image

docker load -i file.tar
